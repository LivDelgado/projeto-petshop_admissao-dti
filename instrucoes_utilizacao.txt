** Instruções para execução do projeto Petshop **

* No terminal (linha de comando) * 

Deve-se acessar o diretório para o qual o arquivo zip foi extraído.
  cd diretorio

Executa-se a seguinte linha:
  java -jar Petshop/dist/Petshop.jar <data> <quantidade de cães pequenos> <quantidade cães grandes>
